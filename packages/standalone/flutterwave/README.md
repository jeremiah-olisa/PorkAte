# @porkate/flutterwave

Flutterwave payment gateway adapter for PorkAte wallet package.

**Status:** ✅ Extraction-Ready (Can be moved to separate repository)

## Features

- 💳 Initialize payments
- ✅ Verify transactions
- 💰 Transfers and payouts
- 🔄 Webhook handling
- 🌍 Pan-African support

## Installation

```bash
npm install @porkate/flutterwave @porkate/payment
```
